CNC Router Development

Problem Statement

Over the years I have enjoyed making experimental guitar pedals and amplifiers but have always been frustrated by how long it takes to solder together circuits on perf boards, and how issue-prone prototyping on breadboards can be. The best way to test such equipment is to order custom PCBs, and then to solder the components in place manually. Unfortunately, ordering custom PCBs is either expensive or requires patience as manufacturing and shipping times can easily add up to multiple weeks. Thus, if I want to be able to prototype my circuits more rapidly, I need to find a way to make PCBs at home.



Overview of PCB Manufacturing Methods

There are two main ways to manufacture PCBs. The industry standard way is to etch copper clad circuit boards using a corrosive solution to remove unwanted copper and a resin mask to shield the wanted copper (the traces) from the corrosive solution. This method also requires the use of a CNC router to cut the board perimeter, drill holes for vias, and mounting holes for through-hole components.

The second method for creating PCBs is to simply use a router to make isolation cuts in the copper cladding to isolate circuit traces. As both methods require a CNC router, and one requires extra steps involving corrosive chemical, I decided to make PCBs using the latter PCB milling method.



Project Overview

  • CNC Router Assembly - An inexpensive & compact cnc router capable of <0.1mm precision in each dimension

  • CNC Motor Controller - An embedded system that receives commands via USB, maintains relative router positioning, and drives stepper motors

  • Router GUI Application - An application that sends path commands to the router & allows user to start, pause/resume, and stop router programs as well as jog the router for relative positioning

  • Image-Based PCB Slicer - A program that takes a screenshot of a PCB & width information, and outputs a path that isolates all of the traces from one another
